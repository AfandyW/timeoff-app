# Timeoff API
## Requirments
- Node 16.13
- Mysql 5.7
## Development
- Install Dependencies and create `.env` file
    ```
    make env
    ```
    Update .env file as your local dev configs
- Run development with docker:
    ```
    make up
    ```
- Stop docker
    ```
    make down
    ```

- Create Database
    ```
    make database
    ```
- Do migrate
    ```
    make migrate
    ```
